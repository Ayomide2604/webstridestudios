# Webstride Studios Website Project

This is the official website rebuild for Webstride Studios, a web development company based in Edmonton, Canada, serving businesses nationwide.

## Company Information
- **Company**: Webstride Studios
- **Website**: www.webstridestudios.ca
- **Location**: Edmonton, Canada
- **Services**: Web Development, UI/UX Design, SEO Optimization, E-Commerce Solutions
- **Contact**: contact@webstridestudios.ca, 437-685-5676

## Project Structure
- **Framework**: Next.js with TypeScript
- **Styling**: Bootstrap-based theme with custom CSS
- **Components**: Header, Footer, Spinner in `/app/components`
- **Assets**: Located in `/public/assets`

## Navigation Structure
- Home (/)
- About (/about)
- Services (/services)
- Portfolio (/portfolio)
- Contact Us (/contact)
